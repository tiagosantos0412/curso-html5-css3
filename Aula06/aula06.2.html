Elementos para melhorar a Semântica

Antes era muito commum a criação de layouts utilizando a tag < table >. Isso hoje em dia é visto como um erro grave, pois a tabela são utilizadas para exibir
dados.

Com o avanço do CSS, as pessoas pararam de usar tabela e a utilizar a tag < div > para criar os layouts, a tag < div > serve para criarmos divisões.
Ela não exibe nada, a menos que a gente estilize com CSS.

O problema é que as pessoas começaram a utilizar a tag < div > para tudo, o que no final das contas não melhorou a semântica em relação a usar table.
