Formulários

Uma necessidade muito comum para desenvolvedores, são locais onde os usuários possam interagir com a página e enviar informações para serem processadas,
como por exemplo, um formulário de cadastro, um formulário de login ou um campo para pesquisas.

Essas formas de se comunicar com a página irão variar de acordo com a necessidade de cada desenvolvedor. Em um determinado site podemos fazer um
cadastro somente com o nome, e-mail e senha, já em outros sites pode haver a necessidade de obtermos várias outras informações.

Iremos então aprender quais as formas que o HTML nos oferece para iteração com os nossos usuários.

Antes de qualquer coisa vamos ver a tag < form >, essa tag não apresenta uma referência visual quando criada, mas apresenta smsnticamente um
formulário, então é dentro dela que vamos criar nossos campos.

A tag < form > pode aceitar vários tipos de atributos, com os quais podemos, por exemplo, configurar como o formulário irá enviar as
informações digitadas para o servidor.

Accept-charset

É utilizado para definir qual charset será utilizado pelo formulário. Por exemplo: UTF-8.

Assim com na tag < a >, o target indica onde a página de destino será aberta.
