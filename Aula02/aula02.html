Listas ordenadas e não ordenadas

Indica uma lista ordenada por números ou letras. Para isso utilizaremso a tag < ol > para criar a lista, e cada item da lista é idicado pela tag < li >

  1. HTML
  2. CSS
  3. JavaScript

Podemos passar um atributo "type" para a tag < ol > para indicar o tipo de ordenação que queremos.

  • type="1" - Lista por números
  • type="A" - Lista por letras maiúsculas
  • type="a" - Lista por letras minúsculas
  • type="I" - Lista por númeos romanos maiúsculos
  • type="i" - Lista por númeors romanos minúsculos

Progress e Meter

Podemos representar o progrsso de algum processo com a tag de progresso. Passamos o valor total do progresso no atributo "max", e o valor do progresso em "value".

Podemos ter algo parecido com a tag < meter >. Também podemos indicar quais valores são altos e baixos com os atributos high e low.

Tabelas

Tabelas serverm para exibir dados tabulados. Para iniciar uma tabela usamos a tag < table >.
Em seguida temos que definir uma linha. Linhas são definidas pela tag < tr >. Dentro de cada linha temos as colunas definidas pela tah < td >.
Opcionalmente, caso queira definir melhor a separação do cabeçalho e o corpo da tabela, você pode utilizar as tags < thead> e < tbody >. Para indicar o título de uma tabela, utilizamos a tag < th > no lugar de < td >.
